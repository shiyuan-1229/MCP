# MCP Forge 便利店智能助手 Demo

## 项目简介
MCP Forge 是一个**零售门店 AI 助手平台**，将传统零售系统通过 MCP 协议标准化为可被 AI 调用的接口，让顾客在微信对话框里**一句话完成查库存、查优惠、下单**。

## 核心能力
- **18 个 MCP 工具**（商品4 + 库存2 + 会员3 + 门店2 + 订单3 + **KBaaS 4**）
- **完整支付闭环**：下单自动生成支付链接，支持微信支付模拟
- **管理后台数据同步**：元器 AI 创建的订单实时同步至管理后台
- **实时数据交互**：查询库存、价格、促销活动
- **多门店支持**：美佳便利店、鲜多多门店数据隔离

## 演示话术
- "可口可乐 330ml 多少钱？"
- "鲜多多今天有哪些优惠？"
- "帮我下个单：可口可乐×2、农夫山泉×1，美佳门店自提"

## 技术亮点
- MCP 协议 + SSE 长连接
- cpolar 公网穿透 + Express + SQLite
- 微信生态无缝接入（小程序/网页）

## 适用场景
- 腾讯校招产品助理岗位答辩
- 毕业设计《基于Web的南国运动场地智能预约与管理系统》延伸
- 中小零售商数字化改造方案验证

> ⚠️ 注意：演示版本使用 cpolar 内网穿透，生产环境建议 CloudBase 部署
