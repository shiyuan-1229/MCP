# 会员门店服务

> MCP Forge 自动生成 · 门店：测试门店B

## 启动方式

```bash
npm install
node server.js
```

## 连接信息

- SSE 端点：http://localhost:4001/sse
- 健康检查：http://localhost:4001/health

## Tool 列表

- member_info: 查询会员信息
- member_points: 查询会员积分及可兑换优惠券
- coupon_query: 查询会员可用优惠券
- store_list: 查询附近门店列表及营业状态
- store_status: 查询指定门店的营业状态
