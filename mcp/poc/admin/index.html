接入配置

🔌 在 WorkBuddy 中使用 MCP 资产

方式一：Tool Call 协议（当前可用）

  1. 在 WorkBuddy 的 ~/.workbuddy/models.json 中配置 AI 模型
  2. 平台提供 API 端点：GET /api/workbuddy/assets 获取资产和 Tool 定义
  3. WorkBuddy AI 自动根据用户意图选择调用哪个 Tool
  4. 调用 POST /api/workbuddy/assets/:id/execute 执行 Tool（POC 阶段为模拟执行）
  5. 调用 POST /api/workbuddy/chat 完成端到端对话（AI 判断→执行→自然回复）

方式二：MCP SSE 协议（规划中）

平台将生成可部署的 MCP Server（SSE 端点），WorkBuddy 通过标准 MCP 协议直连，Tool 真实执行。
