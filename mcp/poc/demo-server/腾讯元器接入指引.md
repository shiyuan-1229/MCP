# MCP Forge · 腾讯元器接入操作指引

> 目标：将 MCP Server 部署到公网 → 在腾讯元器注册为插件 → 创建智能体 → 演示对话
> 耗时：约 30 分钟

---

## 前置准备

- [x] Node.js 22.22.2（已安装）
- [x] 全功能 MCP Server（`demo-server/server.js`，13 Tools，API Key 认证）
- [ ] 腾讯元器账号（https://yuanqi.tencent.com）

---

## 第一步：启动 MCP Server + 公网隧道（2 分钟）

### 方式 A：一键启动（推荐）
双击 `demo-server/start-public.bat`，会同时启动：
1. MCP Server（端口 3456）
2. localtunnel 公网隧道

启动后会显示公网地址，例如：
```
https://fancy-cat-42.loca.lt
```

### 方式 B：手动启动
```bash
cd demo-server
set API_KEY=demo-key-2026
node server.js
# 另一个终端
npx localtunnel --port 3456
```

### 验证
```bash
# 测试健康检查
curl https://你的域名.loca.lt/health
# 应返回: {"status":"ok","name":"retail-store-ai","store":"美佳便利店","tools":13,"auth":true}

# 测试 SSE 端点（需要 API Key）
curl -H "Authorization: Bearer demo-key-2026" https://你的域名.loca.lt/sse
# 应返回 HTTP 200
```

## 第二步：腾讯元器注册 MCP 插件（5 分钟）

1. 打开 [腾讯元器](https://yuanqi.tencent.com) → 登录
2. 进入「插件广场」→「接入 MCP 插件」
3. 填写信息：

| 字段 | 值 |
|------|-----|
| 名称 | 美佳便利店 AI 助手 |
| 描述 | 美佳便利店零售助手：支持商品搜索、库存查询、会员信息、门店管理、订单下单 |
| URL | `https://你的域名.loca.lt/sse` |
| 协议 | SSE (Streamable HTTP) |
| 插件类型 | 自定义插件 |
| 鉴权方式 | API Key |
| API Key | `demo-key-2026` |

4. 点击「保存」
5. 进入「我的插件」→ 点击「测试」验证：
   - 输入 "搜索元气森林" → 应返回商品列表
   - 输入 "美佳便利店有哪些促销活动" → 应返回促销列表

## 第三步：创建智能体应用（5 分钟）

1. 在腾讯元器创建「智能体应用」
2. 基础设置：
   - 名称：美佳便利店小助手
   - 描述：我是美佳便利店的AI助手，可以帮你查商品、看库存、了解促销、下单购物
3. 工具设置 → 添加「美佳便利店 AI 助手」插件
4. 模型选择：混元大模型（推荐）
5. 系统提示词（复制以下内容）：

```
你是美佳便利店的智能助手。你可以帮助顾客：

1. 搜索商品 — 帮顾客找到想要的商品
2. 查询价格 — 告诉顾客商品的价格和规格
3. 查看库存 — 告诉顾客某件商品还有多少库存
4. 了解促销 — 介绍当前有哪些优惠活动
5. 下单购买 — 帮顾客下单并生成取货码

对话规则：
- 先了解顾客需求，再推荐合适的商品
- 查库存时先确认商品名称
- 下单前必须展示商品信息和总价，由顾客确认
- 下单成功后告知取货码和门店信息
- 推荐相关促销活动以促进成交
- 回复简洁清晰，控制在200字以内
```

6. 保存并发布

## 故障排查

### Q: localtunnel 显示 "your url is: ..." 但访问不了？
- 首次访问 localtunnel URL 时会显示密码页面，在浏览器打开填写你的公网 IP 即可通过

### Q: 腾讯元器测试插件返回错误？
- 确认 API Key 填写正确（`demo-key-2026`）
- 确认公网地址末尾带 `/sse`
- 用 curl 单独测试：`curl -H "Authorization: Bearer demo-key-2026" https://你的域名.loca.lt/health`

### Q: 智能体对话时没有调用工具？
- 检查系统提示词是否清晰描述了工具的用途
- 在智能体设置中确认插件已关联
- 用明确的触发词测试（如"帮我搜索元气森林"）

### Q: 演示结束后如何关闭？
- 关闭两个终端窗口即可
- 或 `Ctrl+C` 停止 localtunnel，再关闭 MCP Server 窗口
