# 国内稳定部署指引

## 🚀 方案一：腾讯云开发 CloudBase（最推荐，国内腾讯系）

### 前置条件
1. 注册 [腾讯云](https://cloud.tencent.com/) 账号 → 开通云开发 CloudBase
2. 创建一个云开发环境 → 获取 `envId`

### 部署步骤
```bash
# 安装云开发 CLI
npm install -g @cloudbase/cli

# 登录
cloudbase login

# 配置
# 编辑 cloudbaserc.json 中的 envId 改为你的环境ID

# 部署
cloudbase functions deploy
```

### 部署后获取公网地址
在腾讯云控制台 → 云开发 → HTTP 服务 → 配置 `mcp-server` → 获得 HTTPS 地址

### 腾讯元器配置
- SSE URL: `https://<your-service-id>.service.tcloudbase.com/sse`
- API Key: `demo-key-2026`
- Header: `Authorization: Bearer demo-key-2026`

## 方案二：cpolar 内网穿透（免费版够用，国内稳定）

### 步骤
1. 官网下载：https://www.cpolar.com/ → 注册账号
2. 解压后双击 `cpolar.exe` 授权
3. 认证：`cpolar authtoken <你的authtoken>`
4. 启动隧道：
```bash
cpolar http 3456
```
5. 输出会显示 HTTPS 地址，如 `https://xxx.cpolar.io`

### 腾讯元器配置
- SSE URL: `https://xxx.cpolar.io/sse`
- API Key: `demo-key-2026`

## 方案三：本地演示（最快，无需公网）

如果答辩现场能看到你的电脑，可以直接在本地演示：

1. 保持本地服务器运行：
```bash
cd demo-server
node server.js
```
2. 地址用：
```
http://localhost:3456/sse
```
