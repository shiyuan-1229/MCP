# MCP Forge · 简历项目描述

## 项目概述

**MCP Forge** — 企业级 MCP（Model Context Protocol）打造与治理平台，通过 MCP 协议将企业私有数据和能力封装为 AI 可调用工具，让大模型深度融入企业业务场景。

---

## 档一：3 句话精简版（电梯演讲 / 技能列表）

独立完成企业级 MCP 服务平台的全栈开发，基于 MCP 协议实现 18 个业务工具的 AI 调用封装，支持多租户隔离、微信 Agent 接入和知识库即服务（KBaaS），管理后台覆盖工具工厂、计费管理、网关治理等全链路运营能力。

---

## 档二：1 段标准版（项目经验 / 技能详述）

独立负责 MCP Forge 平台的全栈设计与实现：基于 Node.js + Express 构建 MCP SSE Server，封装 18 个业务工具（商品/库存/会员/门店/订单/知识库），实现多租户隔离（不同商户看到不同工具列表）和 KBaaS 知识库检索（TF-IDF + bigram 分词）；管理后台采用原生 HTML/CSS/JS 开发，区分管理员（运营台：项目工作台/MCP工厂/计费管理/网关治理）与客户（交付门户：MCP资产/数据看板/AI中心）双视角；通过微信 Agent（腾讯元器）实现自然语言交互，结合 cpolar 公网穿透支持外部调用；完成 Docker 容器化部署，支持一键启动生产环境。

## 档三：2 段详细版（项目经验区 / 深度描述）

### 项目背景

MCP Forge 是一个企业级 MCP 打造服务平台，旨在帮助企业将私有数据和业务能力封装为标准化 AI 工具，让大语言模型能够安全、可控地调用企业系统。项目覆盖零售、制造、金融、物业、教育等多行业场景，支持从工具开发、知识库构建到计费运营的全生命周期管理。

### 技术架构与实现

**后端架构**：基于 Node.js + Express 构建双服务架构——主服务（:3100）提供管理后台 API、用户认证、MCP 协议 SSE 端点；Demo 服务（:3458）提供公网 MCP 调用能力。采用 better-sqlite3 实现数据持久化，管理后台与 Demo 服务共享同一 SQLite 数据库确保数据实时同步。实现多租户隔离机制：通过 `mcp_tools` 表和 `mcp_tool_bindings` 关联表，不同商户仅能看到自己被授权的工具，SSE 端点和 API 均按商户过滤。

**MCP 协议层**：基于 `@modelcontextprotocol/sdk` 实现 MCP Server，定义 18 个 Tool（商品查询/库存预警/会员分析/门店运营/订单处理/知识库检索等），支持 JSON Schema 参数校验和调用结果结构化返回。KBaaS 模块采用 TF-IDF 向量检索 + bigram 中文分词，零外部依赖即可实现知识库问答和文档检索。

**管理后台**：原生 HTML/CSS/JS 开发，无框架依赖，支持双角色视角——管理员侧聚焦平台运营（项目工作台、MCP 工厂、Skill 场景包、接入配置、知识库维护、测试与发布、网关治理、计费管理、操作审计），客户侧聚焦业务使用（MCP 资产概览、数据看板、AI 中心、文件下载）。AI 中心覆盖 7 大行业场景（零售/制造/金融/物业/教育/医疗/物流），支持场景切换和意图识别。

**微信生态集成**：通过腾讯元器配置微信 Agent，支持元宝 App 和小程序双渠道发布，用户可通过自然语言调用 MCP 工具完成业务操作。

**部署运维**：完成 Docker 容器化改造，单镜像同时启动主服务和 Demo 服务，通过 docker-compose 实现一键部署，数据持久化挂载独立 volume，健康检查自动验证服务可用性。

### 技术栈

`Node.js` `Express` `SQLite` `MCP SDK` `SSE` `WebSocket` `Docker` `腾讯元器`
